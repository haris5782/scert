-- Persyaratan System
- SLiMS 3 Stable 14 aktif (download di http://slims.web.id/)
- Modul Plugins "Bango" 0.3 aktif (download di http://sutriadi.web.id/)

-- Instalasi
- uraikan paket instalasi scert-x.x.tar.gz di dalam direktori ./admin/modules/plugins/
  sehingga akan menghasilkan direktori ./admin/modules/plugins/scert/
- masuk ke menu Plugins, kemudian aktifkan plugin SCert

-- Hapus Instalasi
- masuk ke menu Plugins, kemudian non-aktifkan plugin SCert
- hapus direktori scert dari direktori ./admin/modules/plugins/ bila perlu

-- Menjalankan Plugin
- masuk ke menu Plugins
- klik link Launch pada baris plugin aktif bernama SCert
