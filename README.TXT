# Persyaratan System
- SLiMS 3 Stable 15 "Matoa" (download di http://slims.web.id/)
- Modul Plugins aktif versi 3.15-2.x "Nantu"
  (download di http://sutriadi.web.id/)

# Instalasi
- uraikan paket instalasi scert-x.x-x.x-x.tar.gz di dalam direktori
  ./admin/modules/plugins/include/ sehingga akan menghasilkan direktori
  ./admin/modules/plugins/include/scert/
- masuk ke menu Plugins, kemudian aktifkan plugin SCert
catatan: bila melakukan instalasi dari paket yang diambil dari github,
silahkan mengubah nama direktori hasil penguraian
dari ./sutriadi-scert-xxx menjadi ./scert

# Hapus Instalasi
- masuk ke menu Plugins, kemudian non-aktifkan plugin SCert
- hapus item scert dari submenu dataTables bila perlu
- hapus direktori scert dari direktori
  ./admin/modules/plugins/include/ bila perlu

# Menjalankan Plugin
- masuk ke menu Plugins
- klik tombol Launch pada baris plugin aktif bernama SCert
